Press A Key to Start

To start the game press any key


Computer'll show you any of the four colours: green, red, yellow, blue


You need to repeat the colour by pushing the same button


After 1 second Computer'll show you the next colour, you need to push buttons in according with whole sequence that you've pushed previously


For example: first colour - red, you need to push the red button, next level - yellow - you need to push red, yellow; next level - green - you need to push red - yellow - green and so on and so forth


If you missed any colour - you'll lose, restart the game by pushing any key


Press "How to play" again to hide the rules of the game
